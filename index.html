PublishHub

AI‑powered local video generator for Reddit stories and Discord memes

⚠ Work in Progress

PublishHub is an early‑stage application currently under active development. The app runs locally on Windows PCs and uses AI to generate videos from Reddit posts, Discord memes, and user‑provided text. No cloud processing is used — all generation happens on the user’s machine.

What PublishHub Does

PublishHub is designed to help creators generate short‑form videos quickly and locally. The app currently supports:

  • ✔ Generating Reddit story videos with AI narration and captions
  • ✔ Posting memes directly from Discord servers
  • ✔ Fully offline processing — no data leaves your device
  • ✔ Automatic captioning, timing, and video assembly

Current Development Status

The core functionality is working, and the app is usable in its early form. The following components are currently in progress:

  • • User interface improvements
  • • Legal documentation and compliance
  • • Homepage and website structure
  • • Packaging and installer creation

TikTok Posting Integration

TikTok posting is already implemented inside the app, but it will remain locked until the required API review and compliance steps are completed. Once approved, TikTok posting will be available to all users.
